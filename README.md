# 政治シミュレーションゲーム - 選挙で未来を変えよう

## 概要

これは、若者に選挙への参加を促すことを目的とした政治シミュレーションゲームです。日本の社会情勢や政治問題を題材に、プレイヤーの選択と投票行動が国の未来を大きく変えることを体験できます。

**重要なメッセージ**: 若者が選挙に行かなければ、何も変わりません。

## 特徴

- **20種類以上の多様なエンディング**: 「ED1: ファシズムの台頭」など、プレイヤーの選択によって様々な未来が訪れます
- **政治・社会問題に基づいたイベント**: 選挙、経済危機、福祉改革、SNSキャンペーンなど15種類以上の実際的なイベント
- **4つの主要パラメーター**: 民主主義、経済、社会福祉、若者の関心を管理
- **投票率の変動**: プレイヤーの行動が若者の投票率に影響
- **確率的イベント発生**: 毎回異なるゲーム体験
- **バッドエンドからハッピーエンドまで**: 様々な結末を通じて政治参加の重要性を学習

## プレイ方法

1. ゲームページにアクセス
2. 「ゲームを始める」ボタンをクリック
3. 10ターンの間、様々なイベントに対して選択を行う
4. 選択によって投票率、民主主義、経済、福祉、若者の関心度が変化
5. ゲーム終了時に、これまでの選択に基づいたエンディングが表示される

## GitHub Pages でのデプロイ

このゲームは静的HTMLサイトとして実装されており、GitHub Pagesで簡単にデプロイできます。

### デプロイ手順

1. リポジトリの Settings > Pages に移動
2. Source を "Deploy from a branch" に設定
3. Branch を "main" (または "master") に設定し、フォルダーを "/ (root)" に設定
4. Save をクリック
5. 数分後、`https://[username].github.io/Political-Game/` でアクセス可能になります

## ゲームの目的

このゲームは単なるエンターテインメントではありません。以下の重要なメッセージを伝えることを目的としています：

1. **投票の重要性**: 若者の投票が社会を変える力を持つ
2. **政治参加の影響**: 無関心は悪い結果につながる可能性がある
3. **民主主義の価値**: 民主主義を守るには市民の参加が必要
4. **社会問題への関心**: 経済、福祉、環境などの問題は若者の未来に直結
