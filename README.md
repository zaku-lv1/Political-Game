# 政治シミュレーションゲーム - 選挙で未来を変えよう

## 概要

これは、若者に選挙への参加を促すことを目的とした政治シミュレーションゲームです。日本の社会情勢や政治問題を題材に、プレイヤーの選択と投票行動が国の未来を大きく変えることを体験できます。

**重要なメッセージ**: 若者が選挙に行かなければ、何も変わりません。

## 特徴

- **28種類の多様なエンディング**: 「ED1: ファシズムの台頭」から「ED28: 燃え尽きた活動家たち」まで、プレイヤーの選択によって様々な未来が訪れます
- **複雑な選択システム**: 各イベントに3〜4つの選択肢があり、それぞれにトレードオフが存在
- **日本の選挙制度を反映**: 小選挙区比例代表並立制、国民投票、連立政権など、実際の選挙制度に基づいたイベント
- **世界情勢イベント**: 国際貿易、気候変動サミット、地政学的緊張など、グローバルな視点のイベント
- **21種類以上の政治・社会イベント**: 選挙、経済危機、福祉改革、SNSキャンペーン、デジタル化など多様なイベント
- **4つの主要パラメーター**: 民主主義、経済、社会福祉、若者の関心を管理
- **投票率の変動**: プレイヤーの行動が若者の投票率に影響
- **確率的イベント発生**: 毎回異なるゲーム体験
- **複雑な結末**: バッドエンド、ニュートラルエンド、ミックスエンド、ハッピーエンドまで、様々な結末を通じて政治参加の重要性を学習

## イベントの種類

### 選挙・政治参加イベント
- **選挙の日** - 投票するか、白票を投じるか、棄権するか
- **比例代表選挙** - 小選挙区比例代表並立制での戦略的投票
- **連立政権の形成** - 連立交渉への市民参加
- **地方選挙** - 期日前投票や候補者比較
- **選挙区割り見直し** - 一票の格差是正への取り組み
- **国民投票法改正** - 直接民主主義の拡充
- **若者枠導入の議論** - 世代間の公平性

### 社会・経済イベント
- **経済危機** - 雇用対策、スキルアップ、起業支援
- **増税案** - 税の使途透明化、若者減税
- **社会福祉改革** - パブリックコメント、署名活動
- **年金問題** - 制度改革への参加
- **教育改革** - 教育政策への意見表明

### メディア・情報イベント
- **SNSキャンペーン** - フェイクニュース対策
- **フェイクニュース** - ファクトチェック、メディアリテラシー
- **メディアリテラシー講座** - 情報の見方を学ぶ

### 市民活動イベント
- **若者デモ** - デモ参加、オンライン支援
- **政治スキャンダル** - リコール運動、説明責任
- **ボランティア活動** - 地域課題への取り組み
- **新しい政党** - 若者政党への支持

### 世界情勢イベント
- **国際貿易協定** - 経済成長と国内産業保護のバランス
- **世界気候サミット** - 環境対策と経済のトレードオフ
- **地政学的緊張** - 外交と防衛のバランス
- **デジタル化政策** - イノベーションとプライバシー保護
- **気候変動政策** - 科学的根拠に基づく政策形成

## プレイ方法

1. ゲームページにアクセス
2. 「ゲームを始める」ボタンをクリック
3. 10ターンの間、様々なイベントに対して選択を行う
4. 各イベントには3〜4つの選択肢があり、それぞれ異なる効果とトレードオフがある
5. 選択によって投票率、民主主義、経済、福祉、若者の関心度が変化
6. ゲーム終了時に、これまでの選択に基づいたエンディングが表示される
7. 同じような選択をしても、イベントの出現順によって結果が変わる

**ヒント**: 
- 完璧な選択は存在しません。すべての選択にはトレードオフがあります
- バランスを取ることも、特定の方向性を重視することも、どちらも有効な戦略です
- 戦略的投票や制度理解など、実際の政治参加に役立つ知識も学べます

## GitHub Pages でのデプロイ

このゲームは静的HTMLサイトとして実装されており、GitHub Pagesで簡単にデプロイできます。

### デプロイ手順

1. リポジトリの Settings > Pages に移動
2. Source を "Deploy from a branch" に設定
3. Branch を "main" (または "master") に設定し、フォルダーを "/ (root)" に設定
4. Save をクリック
5. 数分後、`https://[username].github.io/Political-Game/` でアクセス可能になります

## ゲームの目的

このゲームは単なるエンターテインメントではありません。以下の重要なメッセージを伝えることを目的としています：

1. **投票の重要性**: 若者の投票が社会を変える力を持つ
2. **政治参加の影響**: 無関心は悪い結果につながる可能性がある
3. **民主主義の価値**: 民主主義を守るには市民の参加が必要
4. **社会問題への関心**: 経済、福祉、環境などの問題は若者の未来に直結
